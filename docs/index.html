지금 내 움직임 보기

먼저 내 몸과 마음 상태를 확인하고, 지금 볼 카드와 도움이 될 팩을 하나씩 고릅니다.

STEP 0

지금 상황 확인

지금 내 상태를 가볍게 확인하고, 바로 해볼 만한 작은 행동까지 이어 봅니다.

오늘의 흐름

컨디션 먼저 확인 → 지금 막힌 상태 고르기 → 해볼 카드 고르기 → 하나만 해보기 → 도움이 될 팩 보기

STEP 1

컨디션 먼저 확인

지금 막힌 이유를 보기 전에, 잠·식사·물·몸 상태가 먼저 무너진 건 아닌지 가볍게 확인해요.

해당되는 항목을 체크한 뒤 다음 흐름을 봅니다.
STEP 2

지금 막힌 상태 고르기

아래 문장 중 지금 제일 가까운 것을 하나만 골라 보세요.

STEP 3

지금 해볼 카드 고르기

아래 카드는 지금 상태에서 먼저 해볼 만한 후보입니다. 전부 다 할 필요 없이, 지금 제일 쉬워 보이는 것 하나만 골라 보세요.

3개가 모두 정답이라는 뜻은 아니에요. 지금 가장 해볼 만한 카드 하나만 고르면 됩니다.
